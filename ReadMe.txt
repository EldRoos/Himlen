Himlen:

v0.1 2018-08-12
En uppgift för att visa kunskap om API:er och React.

Installation:

	* Koppiera ner alla filer från projekter till något valfritt lämpligt ställe.

	* Installera Node.js genom att clicka på filen: node-v10.7.0-x64.msi (eller https://nodejs.org/en/download/).
	Andra operatisvsystem: https://www.tutorialspoint.com/nodejs/nodejs_environment_setup.htm

	* Start cmd och navigera till den sparade mappen.

	* Skriv "npm start" för att se sidan i default webbläsare (som får url: http://localhost:3000/).

	* Acceptera Node.js får access.



Instruktioner:

	* Dra i blåa markören för att byta position.

	* Tryck på en dag för att se noggrannare information.
	


Brister:

	* APIn för att se sol uppgång och nedgång har stor fördröjning. Bör bytas till server som är mer lokal.
	
	* När man drar i markören på kartan så hoppar den ibland tillbaka till senaste position.
	
	* Ingen installation utan alla filer är incheckade - och de är många. Borde ändras till installations instruktioner istället.
	
	* En readme för Engelska också.
	
	* Kan flytta utanför Skandinavien på kartan.
